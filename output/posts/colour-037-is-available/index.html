    Colour Developers

    Colour 0.3.7 is available!

    This release is a maintenance release with various fixes, improvements, new objects and backward incompatible changes. It provides support for Colour - Demosaicing and Colour - HDRI.

    The API documentation is now hosted on readthedocs.io

    Python 2.7 and 3.5 wheels are now available on Pypi.

    Head to the releases page for complete details.

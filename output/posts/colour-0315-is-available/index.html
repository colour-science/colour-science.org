    Colour 0.3.15 is available!

    This is mainly a maintenance release implementing support for Python 3.8, Numpy 1.18 and fixing a major issue in Krystek (1985) correlated colour temperature computation.

    Besides the various fixes, the highlights of that release are:

    • Support for Academy Spectral Similarity Index (SSI)

    • Spectral distributions for the CIE LED Illuminants LED-B1, LED-B2, LED-B3, LED-B4 LED-B5, LED-BH1, LED-RGB1, LED-V1, LED-V2.

    • Spectral distributions for the CIE Indoor Illuminants ID65 and ID50.

    Head to the releases page for complete details.

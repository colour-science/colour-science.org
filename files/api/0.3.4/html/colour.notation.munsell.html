colour.notation.munsell Module¶

Munsell Renotation System¶

Defines various objects for Munsell Renotation System computations:

  • munsell_value_Priest1920(): Munsell value \(V\) computation of given luminance \(Y\) using Priest, Gibson and MacNicholas (1920) method.
  • munsell_value_Munsell1933(): Munsell value \(V\) computation of given luminance \(Y\) using ⁠Munsell, Sloan and Godlove (1933)⁠ method.
  • munsell_value_Moon1943(): Munsell value \(V\) computation of given luminance \(Y\) using Moon and Spencer (1943) method.
  • munsell_value_Saunderson1944(): Munsell value \(V\) computation of given luminance \(Y\) using Saunderson and Milner (1944) method.
  • munsell_value_Ladd1955(): Munsell value \(V\) computation of given luminance \(Y\) using Ladd and Pinney (1955) method.
  • munsell_value_McCamy1987(): Munsell value \(V\) computation of given luminance \(Y\) using McCamy (1987) method.
  • munsell_value_ASTMD153508() [1] [2]: Munsell value \(V\) computation of given luminance \(Y\) using ASTM D1535-08e1 (2008) method.
  • munsell_colour_to_xyY() [1] [2]
  • xyY_to_munsell_colour() [1] [2]

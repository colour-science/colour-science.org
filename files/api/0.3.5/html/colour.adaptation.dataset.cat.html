colour.adaptation.dataset.cat Module¶

Chromatic Adaptation Transforms¶

Defines various chromatic adaptation transforms (CAT):

  • XYZ_SCALING_CAT: XYZ Scaling CAT [1]
  • VON_KRIES_CAT: Von Kries CAT [1]
  • BRADFORD_CAT: Bradford CAT [1]
  • SHARP_CAT: Sharp CAT [4]
  • FAIRCHILD_CAT: Fairchild CAT [2]
  • CMCCAT97_CAT: CMCCAT97 CAT [5]
  • CMCCAT2000_CAT: CMCCAT2000 CAT [5]
  • CAT02_CAT: CAT02 CAT [3]
  • CAT02_BRILL_CAT: Brill and Süsstrunk (2008) corrected CAT02 CAT [6] [7]
  • BS_CAT: Bianco and Schettini (2010) CAT [4]
  • BS_PC_CAT: Bianco and Schettini PC (2010) CAT [4]

colour.characterisation.dataset.colour_checkers.chromaticity_coordinates Module¶

ColourCheckers Chromaticity Coordinates¶

Defines ColourCheckers chromaticity coordinates in CIE xyY colourspace.

Each ColourChecker data is in the form of a list of 24 namedtuples as follows:

[('index', 'name', 'x', 'y', 'Y'), ..., ('index', 'name', 'x', 'y', 'Y')]

The following ColourCheckers data is available:

  • BABELCOLOR_AVERAGE: Average data derived from measurements of 30 ColourChecker charts.
  • COLORCHECKER_2005: Reference data from GretagMacbeth published in 2005.
  • COLORCHECKER_1976: ColourChecker developed by McCamy and Al. at Macbeth, a Division of Kollmorgen.
